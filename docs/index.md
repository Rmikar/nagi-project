# 凪プロジェクト

凪は、生活、記憶、土地、文化、関係、自然、時間の積層から世界を捉える思想です。

非所有、共鳴、呼吸を手がかりに、所有と支配だけを社会の原理にしない未来を考えます。

- [自由・不協和・離脱の原則](https://rmikar.github.io/nagi-project/freedom_and_dissent.html)
- [記憶・同意・忘却の憲章](https://rmikar.github.io/nagi-project/memory_and_consent.html)

## 凪の約束

- 共鳴を、人の価値を決める点数にしない。
- 基礎的な生活を、参加やAI利用の条件にしない。
- AIを使わない、記録を残さない、途中で離れる自由を守る。
- AIを、人の権利や生活を最終決定する中心にしない。
- 記憶と継承には、同意と忘却の自由を置く。
