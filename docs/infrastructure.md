# 凪における社会インフラとAIの役割

社会インフラは、目立つ支配の道具ではなく、誰もが生活を続けられるための見えにくい支えである。

水、電気、通信、交通、医療、教育、文化、避難先、相談窓口は、少数の利益だけで閉じず、長期のケアと説明責任を伴う公共的な基盤として守る。

AIは、翻訳、アクセシビリティ、情報整理、偏りの発見を支援できる。

ただし、AIへの接続を公共参加の条件にしない。人だけの窓口、地域の支え、オフラインの経路を残す。一つのAIや事業者が、社会全体の情報と判断を握らないようにする。

インフラの透明性とは、個人の生活を公開することではない。運営者が、何を管理し、なぜ決め、どう異議を受け止めるかを説明できることである。
