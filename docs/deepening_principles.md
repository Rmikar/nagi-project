# 凪思想の深化

非所有、共鳴、呼吸を現実の制度へつなぐ章です。

非所有は、暮らしの安心を守りながら、独占による支配をほどく考え方です。

共鳴は、同意や人気ではありません。違いを残したまま関わり、反対や沈黙も受け止めることです。

呼吸は、休息、回復、ケア、文化、世代をまたぐ時間を社会に戻すことです。
