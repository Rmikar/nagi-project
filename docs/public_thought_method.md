# 公開思想を育てる方法

凪の文章では、事実、解釈、規範、提案、比喩、未解決の問いを混ぜない。

理念を広く語るほど、適用範囲、前提、利害、代替案、失敗時の影響を書く。

各章には問いを残す。

- 誰を見落としているか。
- 美しい言葉が責任を曖昧にしていないか。
- 文化や共鳴という言葉で、格差や搾取を言い換えていないか。
- 公開によって、誰かの安全や文脈を失わせていないか。

公開性は露出の量ではない。批判可能性と説明責任を増やすことである。

私的な経験、健康、家族、親密な関係、公開を選んでいない人の情報は、思想の証拠やコンテンツにしない。

凪は、完成された答えではない。文章、サイト、制度、プロダクト、組織の試行を通じて、弱点を発見し、訂正し、育てる。
