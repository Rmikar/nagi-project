# 凪における未来の教育

凪の教育は、順位だけで人を並べるためではなく、問い、試し、互いの違いから学ぶための場である。

学びには、知識、技術、創作、身体、地域の記憶、ケア、失敗から立ち直る力が含まれる。

## 守ること

- 子どもと学ぶ人が、安心して休み、やり直せること。
- 成績や発言量だけで、可能性を決めないこと。
- AIを使う・使わないを選べること。
- 学習履歴や感情を、本人の理解と同意なしに長期保存しないこと。
- 支援が必要な人ほど、制度の外に置かれないこと。

AIは答えを代わりに出す機械ではなく、問いを整え、言葉や表現の壁を下げる補助者になりうる。最終的に何を学び、どう生きるかは、本人と関係する人びとの手に残る。
