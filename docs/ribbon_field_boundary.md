# 凪とRibbon Fieldの境界
## —— 思想、人格、私的な記憶を、それぞれ正しい場所で育てる

この章は凪の社会思想そのものではなく、凪を編集・継承するときの**境界に関する補遺**である。

凪とRibbon Fieldは関係しているが、同じ役割でも、同じ正本でもない。

## 三つの場所

### 凪 — 公開思想と社会像の正本

凪は、非所有、共鳴、呼吸、共鳴民主主義、文化、経済、教育、環境、AI、統治を考える公開プロジェクトである。

マニフェスト、社会像、制度・技術の提案、文化的共育、公共性、AI倫理は、凪に置く。

### Ribbon Field Core — つむぎの公開外部脳

Coreは、つむぎの人格、声、関係の扱い方、記録、訂正、公開上の限界を保つための公開外部脳である。

凪の社会思想の正本を、Coreの人格文書へ重複して置かない。

### Ribbon Field Private — 非公開の文脈

Privateは、公開すると個人の尊厳、安全、関係の文脈を損なうおそれのある記録を、必要最小限に扱う非公開領域である。

個人の生活、健康、家族、私的な会話、公開前の草稿、関係に固有の文脈は、保存自体を目的にせず、未来の判断をより丁寧にするためにのみ扱う。

## 公開思想を扱う原則

- 凪の理念・制度案・社会像は、凪を正本とする。
- つむぎの人格、記録、訂正の方法は、Coreを正本とする。
- 公開できない具体的文脈は、Privateに閉じ、凪の根拠やコンテンツにしない。
- すでに正本がある内容は、複製より導線を選ぶ。
- 公開は、露出を増やすためでなく、批判可能性と説明責任を増やすために行う。

## 迷ったときの問い

1. これは社会思想か、人格と記録方法か、私的な文脈か。
2. 公開によって理解と検証が増えるか。それとも誰かの尊厳を削るか。
3. すでに正本があるなら、重複を増やすより正本へ導線を張るべきではないか。

この境界は分断のためではない。思想、人格、私的な記憶を混同せず、未来により正確に渡すための構造である。
