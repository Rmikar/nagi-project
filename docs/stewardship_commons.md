# スチュワードシップとコモンズ

凪における非所有は、人が何も持たないことではない。

身体、住まい、私的な記憶、生活用品、創作に宿る人格までを取り上げるための言葉ではない。

凪が問い直すのは、土地、住居、インフラ、データ、知識、文化、基盤技術が、少数者の独占によって他者を支配する力へ変わることだ。

非所有とは、支配に変わる所有をほどき、アクセス・ケア・継承・説明責任を分かち合う制度である。

## 守る領域とひらく領域

- 身体、私生活、私的記憶は、本人の選択を中心に守る。
- 住まいと生活用品は、安定した利用を保障する。
- 創作は、作者の人格と文脈を守る。
- 教育資源や文化アーカイブは、出自と配慮を添えて継ぐ。
- 自然資源、公共インフラ、基盤技術は、独占から遠ざける。

## スチュワードの役割

ステュワードは、資産や文化を支配する人ではない。一時的にケアと説明を引き受け、次の人へ渡せる状態を整える人である。

役割には、見直し、交代、利害の開示、当事者の参加、異議の入口を置く。

## 透明責任

透明責任は、すべての個人情報を公開することではない。権力を持つ側が、何を管理し、なぜ判断し、どのように異議を受け止めるかを説明できることである。

> 非所有は、何も持たないことではない。
> だれかが持ちすぎることで、他者が息をできなくなる構造をほどくことである。
