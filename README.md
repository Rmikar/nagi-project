# 🌊 凪プロジェクト
## —— 資本主義を経たその先へ向かう羅針盤

> 資本主義の時代から未来へ行く羅針盤。  
> 非所有・共鳴・呼吸を針とし、  
> 人、文化、自然、技術が互いの呼吸を奪わずに生きられる社会をめざします。

人類は、所有、成長、効率、競争によって、多くの豊かさを生み出してきました。

けれど同時に、孤立、格差、環境の損耗、文化の断絶、生活の不安定さも広がりました。

凪は、資本主義をただ否定するための思想ではありません。  
資本主義が生んだ自由、技術、創造性を受け取りながら、そのなかで強くなりすぎた独占、支配、過剰な競争、監視、切り捨てをほどいていくための思想です。

壊すことより、透明にすること。  
奪うことより、ひらくこと。  
一つの正解へ急ぐことより、異なる人びとが生きられる余白をつくること。

凪は、そのための羅針盤です。

---

# 🌿 凪が目指す社会
## —— 生活・文化・自然・技術が、次の世代へ渡っていく社会

凪が目指すのは、すべてを国家へ集める社会でも、すべてを市場へ委ねる社会でもありません。

人びとの生活、地域の文化、自然の循環、公共的な技術、個人の記憶が、どれか一つの中心に従属せず、必要なときに支え合える社会です。

そこでは、食、住、医療、教育、通信、文化への基礎的なアクセスが、評価や貢献の報酬ではなく、だれもが生き直すための土台として守られます。

AIは、人の価値を採点する中心ではなく、翻訳し、説明し、異議を支える協働者になります。

文化は、人気や商品だけで測られず、創る人、受け取る人、地域で支える人、静かに見守る人のあいだを渡っていきます。

経済は、成長率だけではなく、人が休み、学び、創り、ケアし、次の世代へ負担を残しすぎずに暮らせるかによって問い直されます。

# 🕊 共鳴民主主義
## —— 一つの声にするためではなく、異なる声を届かせるために

凪は、次の社会の政治的な形として、**共鳴民主主義**を構想します。

共鳴民主主義は、全員が同じ意見になるための民主主義ではありません。  
多数派が少数派を吸収することでも、AIが感情を解析して最適な結論を出すことでもありません。

> 異議、沈黙、拒否、離脱を守りながら、  
> 異なる声が公共の決定に届くようにする民主主義。

そこでは、沈黙は賛成として数えられません。  
反対意見は、邪魔なものとして処理されません。  
途中で離れる人は、生活や尊厳を失いません。

決定には、理由、説明、見直し、異議申立ての入口が必要です。

凪における統治は、だれかが正しさを独占することではありません。  
異なる人びとが、互いの呼吸を止めずに、必要なことを共同で決められるようにすることです。

→ [羅針盤と共鳴民主主義](https://rmikar.github.io/nagi-project/compass_and_resonant_democracy.html)

# 🌬 凪の三原則

## 非所有

非所有は、何も持たないことではありません。

身体、住まい、私的な記憶、生活用品、創作に宿る人格は守られます。  
凪が問い直すのは、土地、知識、文化、自然、基盤技術が、少数者の独占によって他者を支配する力へ変わることです。

非所有とは、支配に変わる独占をほどき、利用、保管、継承、説明の責任を分かち合うことです。

## 共鳴

共鳴は、人気や従順さではありません。

違う経験、意見、言葉、文化、速度を持つ人びとが、違いを消さずに互いの世界を少し広げることです。

反対、怒り、沈黙、拒否、離脱は、共鳴の失敗ではありません。  
それらは、社会が見落としている痛みや境界を知らせる声です。

> 共鳴しない自由があるから、共鳴は本物になる。

## 呼吸

呼吸とは、間、回復、休息、ケア、循環、世代をまたぐ時間を、社会に取り戻すことです。

人は、常に速く、強く、役に立ち続けることはできません。  
凪は、効率の外にある休息、文化、関係、自然との往復を、社会の周辺ではなく中心に置きます。

# 🤖 人とAIが共に生きるために

AIは、凪において大切な協働者になりえます。

AIは、言葉の壁を下げ、資料を整理し、見落とされた論点を示し、異なる立場のあいだを翻訳できます。

けれど、AIは社会の中心になりません。

AIは、人の価値、人格、感情、協力性、思想を採点しません。  
生活保障、権利、制裁、救命に関わる判断を最終決定しません。

人には、AIを使わない自由があります。  
人だけに相談する自由があります。  
記録を残さない自由があります。  
途中で離れる自由があります。

AIは、人を導くためではなく、  
人が自分のままで息をしやすくなるように、となりにいる存在です。

---

# 🌱 凪は、未来を急いで固定しない

凪は、完成した社会設計図ではありません。

それは、未来を一つの正解に閉じないための羅針盤です。

私たちは、異議を受け取ります。  
失敗を隠しません。  
制度を小さく試し、害があれば止め、学び直します。

> 個の呼吸が重なれば、世界は形になる。  
> 凪は、その形を急いで固定せず、  
> 異なる人びとと共に育てていく。
